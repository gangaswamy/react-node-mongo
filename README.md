# react-node-mongo
Tutorial

1. perform npm install on both node and react
   cd ../react-node-mongo
   cd node
   npm install
   cd ..
   cd react
   npm install
   
2. create mongodb database called "TasksDB"
   mongo
   use TasksDB
   
3. to run node 
   cd node
   node app.js

4. run reactjs
   cd react
   npm start
